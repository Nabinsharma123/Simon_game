Here's how to play:

1.Press any key to start the game. Game will start at level 1 and computer press a button.

2. Now, it's your turn. You have to press the same button. After that Level 1 became Level 2 and the control goes back to computer and computer press another button.

3. Now this time you have to press previous button and current button with same order.

4. Again control goes back to computer and the game will go on like this.

5.Don't forget to remember which order computer press button .

6.Just match computer's moves in the exact order.

7.The game will keep track of button presses.

8.You loose if you did it incorrectly or if you lost.

9.Losing starts the game back at level 1.

10. Matching computer moves correctly advances you to the next level.
